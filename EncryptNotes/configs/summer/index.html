YUM（全称为 Yellow dog Updater, Modified）是一个在Fedora和RedHat以及CentOS中的Shell前端软件包管理器。基于RPM包管理，能够从指定的服务器自动下载RPM包并且安装，可以自动处理依赖性关系，并且一次安装所有依赖的软件包，避免了手动安装的麻烦(寻找资源、下载；放到指定目录安装；处理依赖关系并下载依赖关系的包进行安装)。所以用yum安装，实质上是用RPM安装，所以RPM查询信息的指令都可用。
rpm命令

安装

rpm -ivh ***.rpm
其中i表示安装，v表示显示安装过程，h表示显示进度
更新

rpm -Uvh ***.rpm
升级rpm包
卸载

rpm - e PACKAGE_NAME
删除软件包
rpm - e –nodeps PACKAGE_NAME
删除软件包，不考虑依赖包
rpm -e –allmatches PACKAGE_NAME
删除所有跟PACKAGE_NAME匹配的所有版本的包
查询

rpm - q PACKAGE_NAME
查询软件包
rpm -qp *** .rpm
获取当前目录下的rpm包相关信息
rpm -qa | less
列出所有已安装的软件包
rpm -qf /usr/sbin/httpd
查看某个文件属于哪个软件包，可以是普通文件或可执行文件， 跟文件的绝对路径
rpm - qi PACKAGE_NAME
列出已安装的这个包的标准详细信息
rpm - ql PACKAGE_NAME
列出rpm包的文件内容及所在路径
rpm -q –scripts kernel | less
列出已安装rpm包自带的安装前和安装后脚本


yum 命令

安装程序包

yum install tree
安装 tree 程序
更新

yum update tree

卸载

yum remove tree 或 yum erase tree

查询

yum info tree
查看程序包信息
yum list installed
查看已安装的程序列表
yum list updates
查看可更新的程序列表
